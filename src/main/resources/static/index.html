Group5

手順

  1. プレイヤーは用意されたユーザー(user1~user4)で指スマにログイン，人数がそろえばstartが押せる状態になるのでクリックをする．
  2. ログインした順番で親が決まる．
    プレイヤーは自分の上げる手を選択する．
    親はそれに加え、全員で何本の指が上げられているかの予想を入力する。
  3. プレイヤー全員が上げる手を選択し終われば, ページを更新することで結果が表示される．
    親の予想と合計が同じ場合は「あたり」と表示され親の残りの手の数が1減少する．
    また，違った場合は「はずれ」と表示されて残りの手の数に変化は起きない．
    確認を終えたら「次へ」をクリックする．
  4. 親が変わり次にログインしたプレイヤーに親の専用画面が表示される．
    以降は1~3の工程をプレイヤーの中で誰かの手が0になるまで繰り返す．
  5. 親の残りの手の数が0になると勝敗が表示される．(この時にしっかり喜びを表現して盛り上げる大人の対応を行いましょう)
  6. 勝者を褒めちぎったら，ログアウトしてゲーム終了．
